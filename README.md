## Reminders

- Create requirements.txt:

```bash
pip install pipreqs

pipreqs /path/to/project
```

Or Using Virtual Environment:

```bash
pip install virtual env
mkdir Environments
cd Environments
virtualenv NAME
source NAME/bin/activate
pip freeze --local
```
